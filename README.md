# SmartPocket — Student Budgeting App

A working prototype of SmartPocket, a budgeting app for university students 
living on a fixed monthly allowance. Built from my own UX designs: generated
with Lovable, customized in Cursor with AI help, with a Supabase backend.
As part of a product internship project.
Features: login/sign up, allowance and budget setup, expense tracking by category,
spending breakdown, savings goals, and expense history.

The research, market analysis, and analytics dashboards behind this app:
https://github.com/zainabwaqar1/smartpocket-internship
